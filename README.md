## Instlation 


### Prerequisites
- Git (to clone the repository)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/rimasdai/Vault.git
   cd Vault
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the application (if required):
   ```bash
   npm run build
   ```

4. Run the application:
   ```bash
   npm start
   ```
